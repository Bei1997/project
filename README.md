## 功能描述 (features)：

### 註冊/登入/登出

- 註冊時，使用者可以創建帳號，並設定名稱、信箱與密碼等...
- 登入時能辦別使用者跟管理者，分別導向不同頁面
- 使用者登入後，能夠進行桌邊點餐

### 前台

- 使用者能查看詳細餐點介紹
- 使用者透過頁面點餐、送出菜單
- 使用者查詢歷年訂單
- 使用者查詢會員資料

### 後台

- 管理者只能透過現有管理者帳號登入後註冊新管理者
- 管理者查詢所有管理者資料
